# Creator Onboarding Guide

This guide helps creators set up a tip page and start accepting tips on Base.

## Prerequisites

- A wallet (MetaMask, Rabby, Coinbase Wallet, WalletConnect-compatible wallet)
- Base network enabled in wallet
- Small amount of ETH on Base for withdrawal gas

## 1. Create Your Page

1. Open `/create`.
2. Click `Connect Wallet`.
3. Complete the setup form:
   - Display name
   - Description
   - Suggested amounts
   - Theme
4. Save and generate your creator link.

## 2. Share Your Links

After setup, copy and share:

- Hosted page: `/tip/<your-creator-id>`
- Embed URL: `/embed/<your-creator-id>`
- Badge markdown for README/social profiles

## 3. Receive Tips

Supporters can:

1. Open your hosted page or embed widget.
2. Connect wallet.
3. Select an amount and optionally leave a short message.
4. Confirm transaction on Base.

## 4. Withdraw Funds

Tips accumulate in the protocol balance for your wallet address.

1. Connect the creator wallet.
2. Click `Withdraw`.
3. Confirm transaction.

Funds are non-custodial and withdrawn directly onchain.

## Best Practices

- Use a clear display name and short description.
- Keep suggested amounts simple (for example, small/medium/large).
- Test your page using a second wallet before sharing publicly.
- Save your creator ID in case you need to regenerate embed snippets.
