---
name: explore
description: Explore codebase structure for a feature area before implementation
---

Use an Explore agent to map out the codebase for the given feature area.

The agent should find:
- All related files (source, tests, config, types)
- Key types/interfaces and their relationships
- Existing patterns and conventions used
- Entry points and data flow

Report back a concise summary: file list, key types, and patterns found.

After the agent reports, wait for the user's implementation plan before writing any code.
